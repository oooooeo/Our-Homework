# 员工培训进度系统

这是基于仓库中 Obsidian 进度系统思路扩展出的网页原型。

## 当前版本

- 员工端：选择员工、打开培训文章、标记完成。
- 后台主页：查看员工完成率、任务完成率、总体进度条、最近完成记录。
- 新增任务：后台可以新增“培训文章”类任务。
- 实时同步：同一浏览器的多个页面之间通过 `localStorage` 和 `BroadcastChannel` 同步。

## 使用方式

直接用浏览器打开入口页：

```text
training-web/index.html
```

也可以分别打开两个独立页面：

```text
training-web/employee.html
training-web/admin.html
```

如果同时打开两个窗口，一个打开员工端、一个打开后台端，员工端标记完成后，后台端会同步刷新进度。

## 后续接真实后台

当前版本是前端原型，数据保存在浏览器本地。正式用于多人培训时，需要把数据层替换为后端服务和数据库：

- 员工登录
- 任务表
- 完成记录表
- 后台实时订阅或轮询
- 权限控制
